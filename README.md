## 简介
TODOS是一个管理待办事项的Web应用，前端使用了Vue和vuetify， 后端使用了leancloud做数据存储。

## 开发
``` bash
# 克隆项目
git clone https://github.com/iwfan/todos.git
cd todos

# 安装依赖
yarn
# or
npm install

# 启动服务
yarn dev
# or
npm run dev

# 部署
yarn build
# or
npm run build
```
